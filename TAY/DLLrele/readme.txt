Использование файла rele.dll.
1. В VisSim выбирается блок "userFunction" ("DLL-функция" в VisSim 5.0e) и располагается на поле.
2. Щелчком правой кнопкой мыши по блоку вызывается окно подключения DLL-моделей (DLL Properties).
3. В появившемся окне в верхней строке ввода выбирается DLL-файл "rele.dll", в нижней (Base function) указывается функция, соответствующая желаемому типу реле:
	"rele2" - реле с гистерезисом (двухпозиционное),
	"rele3" - реле с гистерезисом и зоной нечувствительности (трехпозиционное),
	"upor" - реле типа "Упор",
	"luft" - реле типа "Люфт".
4. Если подключение прошло нормально, то повторный щелчок правой кнопкой мыши по блоку вызывает окно настроек выбранного реле.

В файле "example.vsm" находится пример использования реле "rele3".
Примечание: Чтобы использовать в модели несколько блоков реле, необходимо для каждого блока создавать собственную копию DLL-файла.
